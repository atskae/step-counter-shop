# Step Counter Shop

Exploring vibe-coding tools...

## Features To Add
### Basic
* Streak
    * Encourage walking everyday
* Send a friend a sticker
* Friend-goal exclusive stickers
    * You and a friend set a step goal (10,000). Only if both of you reach it, then you both get a friend-goal exclusive sticker (you would use 10,000 step balance to accept the sticker).

    * Or a garden theme - shop contains plants and garden supplies
* location-specific items
    * Ex) NYC stickers would be different from Tokyo stickers
    * User can optional share their current location (just while using the app) to open regional items
